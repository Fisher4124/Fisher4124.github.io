> <f-italic>邦名（中文名称）与学名（拉丁文名称），如鸟翼车轮，相须并进，不具其一，必兼丧其二……如无学名记录，则义类不明，即与世界文化隔绝，等于薪柴。如无邦名记录，则传达无具，即与国内文化隔绝，等于饰品。</f-italic>
> ——鍾觀光（1868-1940）

<h2>〇、解释说明</h2>

<ol>
  <li><strong>扩展区汉字</strong>：收录于Unicode基本多文种平面（Basic Multilingual Plane，BMP）之外的中日韩统一表意文字（CJK Unihan），涵盖扩展BCDEFGHIJ…区（下文亦包括尚未编码的汉字）。</li>
  <li><strong>中文正名</strong>：<a href="https://www.iapt-taxon.org/nomen/main.php">ICNafp</a>、<a href="https://www.iczn.org/the-code/the-code-online/">ICZN</a>、<a href="https://www.the-icsp.org/index.php/code-of-nomenclatur">ICNP</a>等命名法规规定，就生物（分类群）而言，原则上只有[<strong>拉丁文</strong>]<strong>学名</strong>（接受名）具有独一有效性，其余名称不论语言均视为“俗名”。而在中国境内流通的汉语名称，至今虽尚无统一共识，但各类正式出版的志书名录（《<a href="https://www.iplant.cn/">中国植物志</a>》《<a href="http://www.zoology.csdb.cn/">中国动物志</a>》“<a href="https://species.sciencereading.cn/">中国孢子植物志</a>”<sup id="fnref:a"><a href="#fn:a" rel="footnote">a</a></sup>《<a href="https://cpsl.sciencereading.cn/#/">中国古生物志</a>》<sup id="fnref:b"><a href="#fn:b" rel="footnote">b</a></sup>等）仍存在朴素的<strong>中文正名</strong>意识，在编纂过程中尝试追求规范的、标准的本土化汉语名称。本文所列的中文名称部分已经罕用，或不再算作“正名”，但不可否认其曾作为<strong>本土通名</strong>加以收录<sup id="fnref:c"><a href="#fn:c" rel="footnote">c</a></sup> 。</li>
  <li><strong>“生僻字”</strong>：虽中文信息化成果喜人，<a href="https://openstd.samr.gov.cn/bzgk/std/newGbInfo?hcno=5664A728BD9D523DE3B99BC37AC7A2CC">GB/T 2312</a>汉字的“生僻”属性已不太明显<sup id="fnref:d"><a href="#fn:d" rel="footnote">d</a></sup>；但因其发布时间较早又曾被广泛采用，部分位于Unicode BMP的汉字并未列入其中仍会带来不便，下文一并列出以资参考。</li>
  <li><strong>表意文字描述字符</strong>（Ideographic Description Character，<strong>IDC</strong>）：用于构成表意文字描述序列（Ideographic Description Sequence，IDS），描述汉字结构。如“⿰木目”=“相”、“⿱木口”=“杏”、“⿵门口”=“问”等。</li>
</ol>

  </ul>
  <li>当前共识：蔷薇科的píng果（<em>Malus</em> sp.）→苹；蕨类杂草pín（<em>Marsilea</em> sp.）→<f-tshyn>𬞟</f-tshyn>。</li>
</ul>

<p>䗴目（† Fusulinida）=<f-tshyn>𰪛</f-tshyn>目=纺锤虫目</p>

<ul>
  <li>该类群的汉名用字流变史比较迷惑<sup id="fnref:1">[<a href="#fn:1" rel="footnote">1</a>]</sup>：</li>
</ul>

<ol>
  <li>
    1927年，李四光取“筳（纺锤）状之虫”之意，创造了<u>左右结构</u>的“䗴”字<sup id="fnref:2">[<a href="#fn:2" rel="footnote">2</a>]</sup>。
  </li>
  <li>
    1944年，郭宗山译介Dunbar的论文<sup id="fnref:3">[<a href="#fn:3" rel="footnote">3</a>]</sup>中首次使用了<u>上下结构</u>的“<f-tshyn>𰪛</f-tshyn>”字<sup id="fnref:4">[<a href="#fn:4" rel="footnote">4</a>]</sup>【<em>应为铸字工失误</em>】。
  </li>
  <li>
    20世纪中后期，以盛金章为代表的学者在相关专著中几乎都使用了<u>上下结构</u>的“<f-tshyn>𰪛</f-tshyn>”字<sup id="fnref:5">[<a href="#fn:5" rel="footnote">5</a>,</sup><sup id="fnref:6"><a href="#fn:6" rel="footnote">6</a>]</sup>【<em>为何？</em>】。
  </li>
  <li>
    21世纪，童金南、杜远生等人编著的教材中也均使用<u>上下结构</u>的“<f-tshyn>𰪛</f-tshyn>”字<sup id="fnref:7">[<a href="#fn:7" rel="footnote">7</a>,</sup><sup id="fnref:8"><a href="#fn:8" rel="footnote">8</a>]</sup>。
  </li>
  <li>
    2007年，有学者主张应以<u>上下结构</u>的“<f-tshyn>𰪛</f-tshyn>”字作为术语正字，并声称李四光所造tíng字便是<u>上下结构</u><sup id="fnref:9">[<a href="#fn:9" rel="footnote">9</a>]</sup>【<em>或与事实相悖</em>，李四光的原始书影为<u>左右结构</u>的“䗴”字<sup id="fnref:10">[<a href="#fn:10" rel="footnote">10</a>]</sup>】。
  </li>
  <li>
    2009年，全国科学技术名词审定委员会公布的《古生物学名词（第二版）》中，“fusulinid”对应的汉文名为<u>上下结构</u>的“<f-tshyn>𰪛</f-tshyn>”字（术语代码：<a href="https://www.termonline.cn/wordDetail?termName=虫筳&amp;subject=4c5b804f26a811ee88b6b068e6519520&amp;base=1">02.0040</a>）<sup id="fnref:11">[<a href="#fn:11" rel="footnote">11</a>]</sup>【<em>是非曲直，难以论说</em>】。
  </li>
</ol>
